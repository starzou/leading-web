leading-web
========

使用最前沿的技术,搭建的前端框架

### 使用最前沿的技术,搭建的前端框架
### 技术栈 : angular, bower, gulp


## Get
```bash  
$ git clone https://github.com/starzou/leading-web  
$ cd leading-web  
```

## Install
```bash  
$ npm install && bower install
```

## Usage
```bash  
$ npm start
```
